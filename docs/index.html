Setup steps

Install and prepare the phone

Set up MuseTrace AI before opening Instagram Draw for a real drawing session.

Install APK

Download MuseTrace-AI.apk from the latest GitHub release, open it on Android, and allow installation from that source if Android asks.

Gemini setup

Open Settings, paste your Gemini API key, tap Save key securely, then test it. Never commit or share API keys.

Accessibility setup

Enable MuseTrace AI in Android Accessibility settings. This allows user-confirmed drawing gestures only.

Overlay permission setup

Allow Display Over Other Apps so the floating controller can stay above Instagram Draw.

Create or import image

Generate with Gemini, import from Gallery, or capture with Camera, then edit and convert into color layers.

Calibrate drawing area

Save the final Draw canvas with four calibration points after opening the same Instagram Draw layout you will use.

Instagram Draw workflow

  1. Open Instagram manually.
  2. Open the target chat manually.
  3. Open Draw mode manually.
  4. Use Add Space manually if available.
  5. Keep Instagram Draw open until drawing is finished.
  6. Select each overlay color manually in Instagram Draw.
  7. Tap Continue in the floating overlay after selecting the color.
  8. Manually send or save in Instagram when drawing is complete.

Add Space workflow

MuseTrace AI does not automatically tap Add Space. If Instagram shows Add Space and you want to use it, tap it manually first, then calibrate the final drawing area.

If Add Space is missing, continue with the normal Draw canvas. MuseTrace AI cannot enable, unlock, force, or add Add Space.

Disclaimer

MuseTrace AI is not affiliated with Instagram or Meta. It works only if Instagram Draw is already available on the user's account, app version, region, and device. It does not log into Instagram, use private APIs, scrape, bypass security, or send messages automatically.
